Detailed Description

The base class for an image filter. This base class will then be able to apply filter to a pixel buffer as the tool is dragged around interactively.

Some filters can operate in place, while others cannot. Subclasses should override this function and return false if the filter cannot operate in place. The default implmentation returns true.
