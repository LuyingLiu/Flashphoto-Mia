Detailed Description

This is the major class of the FlashPhoto application. In the model-view-controller paradigm, we can think of this class as implementing the model for an ImageEditor that could be controlled by various controllers (e.g., interactive mouse and keyboard, command line) and viewed different ways (e.g., interactively with a GUI, with files written to disk).
