Detailed Description

This filter blurs the image by convolving it with an appropriate motion-blurring kernel. It supports four possible blurring directions (North-to-South, East-to-West, Northeast-to-Southwest, and Northwest-to-Southeast) and blur according to the amount specified in the GUI.
