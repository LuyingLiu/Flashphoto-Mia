Detailed Description

Stores an array of ColorData, such as an image that can be drawn to the screen.

The data are stored internally in a 1D array, laid out in row-major order with 4 floats (RGBA) for each pixel in the image. The raw data can be accessed directly with data() and individual pixels can be accessed with get_pixel(x,y).
