Detailed Description

This is the base class for a convolution filter. Every filter "has a" mask. The new color for a pixel depends not only upon the original color of that pixel but also upon the original colors of its neighbors. The Subclasses must define this mask by filling in the CreateMask factory method. This base class will then be able to apply the mask to a pixel buffer as the tool is dragged around interactively.
