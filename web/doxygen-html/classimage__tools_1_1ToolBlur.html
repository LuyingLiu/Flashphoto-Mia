Detailed Description

This tool serves as a mobile version of the blur filter, functions much like the spray can, except with blurring neighboring pixels rather than coloring them (linear falloff).

ColorData image_tools::ToolBlur::LookupPaintColor ( int  x,
int  y 
)
overridevirtual

This function returns the current paint color used with the tool. In most cases, this is the same as the paint_color passed in to the StartStroke() function. However, some tools, like the blur tool, have "special paint" that depends on the underlying pixels in the canvas or some other factors. Thus, this function provides a hook for subclasses to override the default paint color to implement special paint colors.
