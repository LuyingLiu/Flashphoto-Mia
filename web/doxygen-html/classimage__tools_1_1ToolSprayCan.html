Detailed Description

This tool has a linear falloff so that it looks like the paint has been sprayed onto the canvas. It also has the interesting property that it continues to paint even when held stationary so paint continues to accumulate on the canvas the longer you hold it still.

Member Function Documentation

bool image_tools::ToolSprayCan::applies_paint_when_stationary ( )
inlineoverridevirtual

Most tools only apply "paint" when they are moved on the canvas, but some, like a spray can continue applying more paint over time even if they are held in the same place. Subclasses should override this function and return true if the tool allows paint to accumulate even when held stationary. The default implmentation returns false.
